要求：治療量 > 團隊承受傷害 × 0.8

世界樹源力值計算器（數學模型）

源力值計算公式（JavaScript偽代碼）

function calculateSourcePower(quality, starLevel, tier) {
    // 品質基礎值
    const baseValues = { 'R': 15, 'SR': 30, 'SSR': 60 };
    
    // 升星累進值（R級示例）
    const starBonus = {
        'R': [0, 10, 20, 20, 40, 50, 50], // 索引對應星級
        'SR': [0, 0, 20, 20, 40, 50, 50], // SR初始1星
        'SSR': [0, 0, 0, 20, 40, 50, 50]  // SSR初始2星
    };
    
    // 升階累進值（R級示例）
    const tierBonus = {
        'R': [0, 15, 10, 10, 10, 10],
        'SR': [0, 20, 10, 10, 10, 10],
        'SSR': [0, 30, 10, 10, 10, 10]
    };
    
    const base = baseValues[quality];
    const star = starBonus[quality][starLevel] || 0;
    const tierSum = tierBonus[quality].slice(0, tier + 1).reduce((a, b) => a + b, 0);
    
    return base + star + tierSum;
}
計算示例：R級3星+4階位

15(基礎) + 20(3星) + 45(階位1-4累進) = 80源力值

Q: 無課玩家應該優先培養哪些角色？資源如何分配？

A: 基於巴哈姆特玩家經驗與數學模型分析：

  1. 角色優先級：教皇 > 沙織 > 傳龍 > 紫龍 > 白希（聖女）
  2. 資源分配比例：角色培養60% + 世界樹30% + 裝備10%
  3. 抽卡策略：存夠200抽再抽限定池，避免碎片化投入
  4. 放棄角色：果斷放棄奧丁星矢等高需求角色，集中資源
  5. 長期目標：追求黃金聖衣共鳴，而非短期戰力提升

Q: 世界樹系統中，應該優先升級哪個小世界？

A: 根據陣容類型數學化分析：

  • 坦克隊：光之界（前排）> 智慧之界（全體）> 亡者/勇者之界
  • 控制隊：霧之界（中排）> 勇者之界（念攻）> 智慧之界
  • 爆發隊：亡者之界（物攻）> 霧之界（中排）> 智慧之界
  • 數學原則：升級對應主力陣容角色類型與位置的小世界
  • 重置策略：陣容轉換時重置返還材料，重新分配

Q: PVE和PVP陣容是否需要分開培養？

A: 基於資源效率辯證分析：

  1. 無課/微課玩家：以PVE為主，PVP用同一陣容調整技能順序
  2. 中課玩家：培養1個PVE主力 + 1個PVP專用陣容
  3. 重課玩家：分開培養，PVE追求效率，PVP追求克制關係
  4. 數學模型：資源分攤成本 vs 場景專精收益 平衡計算
  5. 實操建議：80%資源投入PVE陣容，20%調整用於PVP

在《聖鬥士星矢：重生2》的高階玩法中，能量管理直接決定戰局走向。許多玩家只注重角色強度，卻忽略了技能釋放順序與能量回復之間的連動關係。本節將深入解析能量循環機制，並提供三套實用配隊方案，幫助你從被動等待轉為主動掌控戰鬥節奏。

**核心能量機制解析**

遊戲中能量主要來源包含：普攻積累（每段攻擊回復固定值）、受擊積累（根據傷害比例轉換）、角色被動技能（如射手座艾奧羅斯擊殺後額外回復）、以及聖衣覺醒加成。經實測，前排坦克因承受多段攻擊，往往比後排輸出更早集滿能量條，這點可作為連招起手的關鍵。建議在設定中開啟「能量條顯示」功能，隨時監控全隊能量狀態。

**速攻爆發循環：雙輔助推進體系**

此體系以天鵝座冰河（控場+能量減益）搭配仙女座瞬（治療+能量增益）為核心。起手式為冰河釋放「鑽石星塵拳」降低敵方能量獲取，接著瞬使用「星雲鎖鏈」為全隊附加「能量汲取」狀態。實戰數據顯示，此組合能讓主力輸出（如獅子座艾奧里亞）的能量回復速度提升約40%，實現每場戰鬥多開1-2次必殺技的壓制效果。需注意此陣型懼怕雅典娜的淨化效果，建議在敵方無驅散角色時使用。

**穩守反擊循環：傷害轉換能量策略**

採用金牛座阿魯迪巴（格擋反傷）與巨蟹座迪斯馬斯克（傷害標記）的組合，能將敵方攻勢轉化為我方能量優勢。當阿魯迪巴開啟「巨型号角」進入反傷狀態時，敵方每次攻擊都會為我方全隊回復微量能量；迪斯馬斯克的「積屍氣冥界波」則會在目標身上附加「靈魂烙印」，使我方對該目標造成的傷害部分轉化為能量。此體系尤其適合消耗戰，在公會BOSS戰中持續輸出表現突出。

**進階操作技巧：能量閾值管理**

高手對決常出現「能量抵消」現象——當雙方同時釋放必殺技時，系統會根據能量溢出值判定技能優先級。建議養成保留關鍵技能的習慣，例如處女座沙加的「天舞寶輪」應在敵方能量達85%左右時預判釋放，利用其沉默效果打斷對手爆發節奏。另可透過角色站位調整受擊目標，讓次要角色承受群攻技能以加速其能量循環，實現輔助角色更快開出保護技能。
